# Heatmap significant features

Function to check the performance of the models that were fitted.

## Usage

``` r
plot_model_sig_features(
  abundance_table,
  metadata,
  contrasts_results,
  color_by,
  cluster_feat = FALSE,
  cluster_samp = FALSE,
  color_vector = NULL
)
```

## Value

The return value, if any, from executing the function.
